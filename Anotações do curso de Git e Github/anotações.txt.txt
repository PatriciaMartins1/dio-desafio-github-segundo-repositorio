Anotações sobre as aulas de git e github
link para download - https://git-scm.com/downloads

Após criar o repositorio ir para o git bash e fazer clonagem do repositorio

git clone 'link do repositorio'

git status 'para ver os status do arquivo'

git commit 'para fazer commit do arquivo'

git commit -m '- m coloca um comentario'

git add . 'adiciona anova pasta criada'


cd 'nome do arquivo' para entrar no arquivo
